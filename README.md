# WP Theme Soli

~Plugin Name: wp-theme-soli~

~Current Version:1.4.1~

Main public-facing WordPress theme for [soli.nl](https://soli.nl).

## Requirements

- WordPress 6.0+
- PHP 8.4+

## Development

### Local environment

```bash
npm install
npm run env:start
```

Local site runs at `http://localhost:8888` (admin: admin/password).

### Testing

```bash
npm run test:e2e
```

### Building a release

```bash
npm run publish
```
